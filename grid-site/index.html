Under the Hood

The Grid Scan Engine

Four scan layers working in parallel — from passive ARP listening to active service probing — giving you total coverage without noisy false positives.

Layer 1
Passive ARP Discovery
Monitors broadcast traffic to build an initial host list with zero active probes. Silent, stealthy, and safe for production networks.
// Zero packets sent — purely passive
arp_listener.on_broadcast(|frame| arp_table.upsert(frame.src_mac, frame.src_ip))
Layer 2
ICMP & TCP SYN Probe
Sends crafted ICMP echo and half-open TCP SYN probes to confirm host liveness and identify operating system stack fingerprints via TTL and window-size analysis.
// SYN probe — no full handshake
syn_probe(host, port: 80) → ttl=128 → Windows
syn_probe(host, port: 22) → ttl=64 → Linux
Layer 3
Service Banner Grab
Connects to open ports and collects service banners, HTTP headers, and TLS certificate metadata. Identifies exact software versions for CVE matching.
// Banner → version → CVE lookup
SSH-2.0-OpenSSH_7.4 → CVE-2023-38408
Apache/2.4.49 → CVE-2021-41773 (CRIT)
Layer 4
Vulnerability Correlation
Cross-references discovered service versions against the NVD CVE database (updated daily). CVSS v3.1 scores, EPSS exploit probability, and affected version ranges.
// CVSS + EPSS scoring
CVE-2021-44228 (Log4Shell)
CVSS: 10.0 | EPSS: 0.971 | EXPLOIT: public
